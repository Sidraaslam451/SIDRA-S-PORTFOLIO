# Sidra Aslam — Personal Portfolio Website

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript.

## 🚀 Live Features

- Animated particle background with connected nodes
- Typed.js rotating text effect
- Scroll-reveal animations on all sections
- Animated skill progress bars & radial charts
- Interactive modals (services, social links, journey timeline)
- Mobile-responsive with hamburger menu
- Cursor glow effect
- Contact form with validation

## 📁 Project Structure

## 🛠️ Technologies Used

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- JavaScript (Vanilla)
- [Typed.js](https://github.com/mattboldt/typed.js/) — typing animation
- [Boxicons](https://boxicons.com/) — icons

## 📦 How to Run

1. Clone or download this repository
2. Make sure all files are in the same folder
3. Open `index.html` in any browser

No build tools or installations required.
